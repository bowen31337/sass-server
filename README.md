####Usages

```bash
git clone git@github.com:bowen31337/sass-server.git

cd sass-server

yarn install (or npm install)

npm run sass-server


```

####Studio Setup

```
Responsive Skin CSS URL : http://localhost:8080/css/main.css

```

####Working SCSS

add your custom scss in 

```
sass/_style.scsss
sass/_variables.scss

````
